# RoboMan Frontend

Профессиональная информационная система для учёта занятий по робототехнике в детских садах.

## Дизайн-система

Проект использует собственную дизайн-систему (Dashdark X): светлая и тёмная темы, фиолетово-синий акцент.

**Документация:** [src/design-system/DESIGN_SYSTEM.md](src/design-system/DESIGN_SYSTEM.md) — правила использования цветов, типографики, отступов. При разработке и рефакторинге стилей обязательно следовать этой документации.

## Технологии

- React 19.2
- React Router 6
- Axios для API запросов
- Recharts для графиков и диаграмм
- CSS Modules для стилизации

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

Приложение откроется по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для production

```bash
npm run build
```

## Конфигурация

По умолчанию API базовый URL: `http://localhost/api`

Для изменения создайте файл `.env` в корне проекта:

```
REACT_APP_API_BASE_URL=http://localhost/api
```

## Роли пользователей

### Владелец (OWNER)
- Полный доступ ко всем данным
- Управление отделами, филиалами, преподавателями
- Просмотр финансовых метрик
- Редактирование и удаление занятий

### Преподаватель (TEACHER)
- Просмотр только своих занятий
- Создание и редактирование своих занятий
- Просмотр своей зарплаты
- Не видит финансовые данные компании

## Основные функции

- **Авторизация**: логин/пароль
- **Дашборды**: аналитика с графиками и KPI
- **Управление занятиями**: создание, редактирование, удаление
- **Управление филиалами**: CRUD операции
- **Управление преподавателями**: просмотр и управление
- **Инструкции**: управление PDF инструкциями
- **Настройки**: настройка параметров расчёта зарплаты

## API Интеграция

Все API запросы идут через `services/api.js` с автоматической обработкой авторизации и ошибок.
